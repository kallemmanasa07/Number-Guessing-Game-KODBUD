# 🎯 Number Guessing Game

A fun and interactive Number Guessing Game built using Python and Tkinter. The application generates a random number between 1 and 100, and the player must guess the correct number. The game provides hints such as "Too High" or "Too Low" to guide the player and keeps track of the number of attempts taken to win.

## Features

* Generate a random number between 1 and 100
* User-friendly graphical interface using Tkinter
* Displays hints:

  * 📈 Too High
  * 📉 Too Low
* Tracks the number of attempts
* Restart Game functionality
* Full-screen mode support
* Keyboard support:

  * Enter key to submit guess
  * Backspace key to delete input
  * ESC key to exit full-screen mode
* Winner banner and success message
* Input validation for numbers between 1 and 100

## Technologies Used

* Python
* Tkinter
* Random Module
